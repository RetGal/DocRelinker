# DocRelinker
DocRelinker helps you relocate an OpenOffice Writer or Microsoft Word document. It copies all related files and preserves the links to those.

It is usefull if you intend to handle over a document containing (absolute) links to related documents, to another person or computer.

## Usage
1) Choose the main document which contains links to other files.

2) Select a destination path.

3) DocRelinker copies your main document to the destination path. All related files will be copied to a sub folder named "related".

4) As your main document now only has relative links to the related files, you can savely copy the whole folder to a location or storage media of your choice.
